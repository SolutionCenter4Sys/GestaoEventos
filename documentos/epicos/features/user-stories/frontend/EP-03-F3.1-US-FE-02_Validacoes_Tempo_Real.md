## 2. User Story

**Como** Participante  
**Quero** Receber feedback visual imediato sobre campos inválidos  
**Para que** Eu possa corrigir erros antes de enviar o formulário

---

## 3. Descrição

Sistema de validações inline com feedback visual:

**Validações:**
- E-mail: formato válido (regex RFC 5322)
- Telefone: formato (XX) XXXXX-XXXX
- CPF: dígitos verificadores válidos

**Feedback Visual:**
- Campo válido: borda verde + ícone checkmark
- Campo inválido: borda vermelha + mensagem de erro
- Debounce 300ms para evitar validações durante digitação

---

## 4. Critérios de Aceite

- [ ] Validação de e-mail em tempo real (debounce 300ms)
- [ ] Validação de CPF em tempo real
- [ ] Mensagem de erro clara abaixo de cada campo
- [ ] Borda verde quando campo válido
- [ ] Borda vermelha quando campo inválido
- [ ] Ícones de feedback (checkmark, X)
