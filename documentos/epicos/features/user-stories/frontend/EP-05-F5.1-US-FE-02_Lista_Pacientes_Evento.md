## 2. User Story

**Como** Organizador, Admin ou Professor  
**Quero** visualizar lista de pacientes modelo vinculadas ao meu evento  
**Para que** eu possa gerenciar participação, visualizar dados e acompanhar documentação pendente

---

## 3. Descrição

Tabela paginada com pacientes modelo do evento:

**Colunas:**
- Nome
- Idade
- Contato (telefone)
- Status Documentos (Completo/Pendente)
- Ações (Visualizar, Editar, Excluir)

**Filtros:** Nome, idade, status documentos  
**Acesso:** Apenas pacientes do evento ao qual usuário tem acesso

---

## 4. Critérios de Aceite

- [ ] Tabela exibe pacientes do evento selecionado
- [ ] Badge "Documentos Pendentes" se faltarem documentos
- [ ] Busca por nome filtra em tempo real
- [ ] Ação "Visualizar" abre modal com dados completos
- [ ] Ação "Excluir" exige confirmação + senha Admin
- [ ] Paginação (20 por página)

---

```typescript
export interface PacienteModeloListItem {
```
